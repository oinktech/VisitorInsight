Visitor Insight

API Overview

Visitor Insight 是一個用於跟蹤網站訪問者的 API。它支持多語言，並提供訪問者的地理位置和統計數據。

API Endpoints

  • Log Visit: /api/{lang}/visit
  • Get Visits: /api/{lang}/visits
  • Get Visit Details: /api/{lang}/visits/{ip}
  • Get Stats: /api/{lang}/stats
  • Blacklist IP: /api/{lang}/blacklist
  • Get Blacklist: /api/{lang}/blacklist
  • Download Visits: /api/{lang}/download_visits

Example Usage

JavaScript Example

            
fetch('https://your-domain.com/api/en/visit', {
    method: 'GET',
    headers: {
        'x-api-key': 'YOUR_API_KEY'
    }
})
.then(response => response.json())
.then(data => console.log(data));

Response Example

            
{
    "message": "Visit logged",
    "ip": "192.168.1.1",
    "location": "New York, NY"
}

Language Support

The API supports both English and Traditional Chinese. You can choose the desired language by modifying the endpoint.
